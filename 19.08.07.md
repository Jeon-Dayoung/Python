**Total 함수**

```python
def total(scores):
   total_score = 0
   for i in scores:
      total_score += i
   return total_score

my_scores = [80, 100, 70, 90, 40]
print(total(my_scores))
```

**Average 함수**
```python
def average(scores):
   avg_score = total(scores) / len(scores)
   return avg_score

my_scores = [80, 100, 70, 90, 40]
print(average(my_scores))
```
